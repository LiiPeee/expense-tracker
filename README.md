# Project Expense Tracker

```
System about expense tracker to show for you where you spend your money, in additional save contacts.
```

## Technology 

```
Typescript, Express, axios, nodemon, Mysql, Prisma
```

## Run

```
npx jest

npm run dev
```
OR
```
yarn dev
```

## Build
```
npm run build
```
or
```
yarn build
```
